# README #

This is a simple program to edit your favorite files.  

### What is this repository for? ###

* Stable version
* v1.01

### How do I get set up? ###

* sudo apt-get install libncurses-dev
* Download repository
* Unpack
* make ; make install; wconfig
* Enjoy!
